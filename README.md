# HW_05_MLOps

## Цель  
Цель проекта - демонстрация методов обеспечение воспроизводимости эксперимента в рамках домашнего задания 5
Проект демонстрирует работу инструментов MLOps: DVC (управление версиями данных), Feast (хранилище признаков), MLflow (отслеживание экспериментов и моделей).

## Инструкция
1 Скачать проект по ссылке [https://github.com/dardashev/HW_05_MLOps].   
2 Проект создан для версии python 3.14.0.  
3 В корне проекта выполнить: `pip install -r requirements.txt`.  
4 Инициализовать dvc: `dvc init` и добавить данные: `dvc add data/raw/iris.csv`.  
5 Чтобы избежать передачи датасета в git, выполнить `git add data/raw/.gitignore data/raw/iris.csv.dvc`.  
6 Для запуска ml-модели выполнить `dvc repro`.  
7 Для просмотра результатов эксперимента выполнить `mlflow ui --backend-store-uri sqlite:///mlflow.db` и открыть [http://localhost:5000] в браузере.  
8 Для передачи признаков в Feast перейти в папку `feature_repo` и выполнить `feast apply`.  
9 Материализовать признаки в онлайн‑хранилище: `feast materialize 2026-01-01T00:00:00 2026-12-31T23:59:59`.  
10 Открыть UI Feast: `feast ui` и перейти по адресу [http://localhost:8888] в браузере.
